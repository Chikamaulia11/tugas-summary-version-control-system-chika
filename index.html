Summary

Version Control System

Pengertian

Version Control System, disebut juga revision control system atau source code management adalah sistem yang mengelola perubahan dari sebuah dokumen program computer, website dan kumpulan informasi lain. Karena sistem ini merekam perubahan yang terjadi pada sebuah dokumen atau sekumpulan dokumen dari waktu ke waktu maka dimungkinkan untuk kembali ke versi tertentu di kemudian hari.

Jenis-jenis

Local Version Control System

Metode version-control yang digunakan oleh banyak orang adalah menyalin dokumen ke folder lain. Pendekatan ini sangat umum karena sangat sederhana, tetapi juga sangat rentan terjadi kesalahan. Sangatlah mudah untuk lupa folder mana yang sedang digunakan dan secara tidak sengaja menulis ke dokumen yang salah atau menyalin dokumen yang tidak dimaksudkan. Untuk menyelesaikan masalah ini, programmer zaman dulu mengembangkan local VCSs yang memiliki sebuah database sederhana yang menyimpan seluruh perubahan terhadap dokumen dalam revision control.

Centralized Version Control System

Masalah besar lainnya yang orang-orang temui adalah mereka perlu berkolaborasi untuk mengembangkan sebuah sistem. Untuk menangani masalah ini, Centralized Version Control System dikembangkan. Sistem ini memiliki sebuah server tunggal yang berisi seluruh versi dari dokumen, dan sejumlah orang yang mengunduh dokumen dari tersebut. Sistem ini memiliki kelebihan dibandingkan VCS lokal, karena semua anggota tim dapat mengetahui perkembangan proyek. Namun, kelemahannya adalah jika server pusat mati atau rusak tanpa cadangan, maka seluruh data bisa hilang dan kolaborasi terhenti sementara.

Distributed Version Control System (DVCS)

Dalam DVCS (seperti Git, Mercurial, atau Darcs), klien tidak hanya mengunduh salinan terbaru dari dokumen; melainkan, menyalin seluruh repositori, termasuk riwayat lengkapnya. Oleh karena itu, jika server mati dan sistem-sistem ini berkolaborasi melalui server tersebut, repositori klien mana pun dapat disalin kembali ke server untuk memulihkannya. Setiap klon adalah cadangan lengkap dari semua data.
